# Sophie Ryan PSY6422 FinalProject

## Death rates from Alzeimer's disease and other forms of dementia's across Europe from 2000-2021

### Overview 

This project analyses the annual death rate from Alzeimers disease and other forms of dementia's from 2000-2021 globally to visualise how death rates from Alzeimer's disease and other forms of dementia's have changed over time and how this has differed across countries in Europe. The analysis is written in R using Quarto to produce a fully reproducible self-contained HTML report.

#### Key features:

-   Data cleaning and wranglings using `tidyverse`

-   Interactive visualisation using `ggplot`, `plotly`

-   Reproducible report via Quarto
